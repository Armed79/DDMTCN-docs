## 关于 DokiMod

DokiMod 是由 imgradeone 初建的平台，目的是为了发展中文 DDLC Mod 圈子，也算是为 Ren'Py 做推广。目前已提供 DDLC 中文 Mod 模板，接下来将提供素材 / 补丁库，满足开发需求。

本站属于粉丝作品，遵循 [Team Salvato IP Guidelines](/ipguidelines)。
